ojo con esto, esto, esta es una imagen 4k que pongo y ajusto, pero no deja de ser 4k y su peso

primero se carga la imagen mas pequeña en pixeles, es la imagen de base

con la etiqueta source dentro de picture se ponen las otras imágenes

dentro de la etiqueta source el atributo media permite ajustar los rangos como en el css, para este caso los min-width se distribuyen en función de los rangos

para identificar las transiciones se va a inspeccionar , se pasa al menor de los aspectos (resolución) y donde dice elements se selecciona network y sale una lista de los elementos cargados

si llega a haver una transición en la pantalla ni se nota a experiencia de usuario

normalmente para imágenes responsivas el width de 100% es lo mas usado
